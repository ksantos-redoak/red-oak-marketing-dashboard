Win Rate by Source & Campaign

YTD · closed dealswon ÷ (won + lost)
!

Read these with sample size in mind. Win rate = won ÷ closed (won + lost). Only Offline (65 closed) and Direct (17) have enough closed deals to be statistically meaningful; the rest are directional — a rate built on 2–6 closed deals moves a lot with one deal. Each bar shows its closed-deal count so nothing is over-read. Campaign win rate is a "touched" influence measure (an opp counts for any campaign a member is on), so a deal can appear under several campaigns — it isn't clean single-attribution.
